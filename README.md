# Random Cat Meme Generator

A fun one-click web app that creates random cat memes using random cat images and meme-style captions.

## Description

Random Cat Meme Generator is a playful web application built with HTML, CSS, and Vanilla JavaScript.  
The app combines a random cat image with a random caption to create a meme in a classic meme format.  
Users can generate a new meme instantly and download the final image to share with others.

## Features

- Random cat image generator
- Random caption generator
- Classic meme-style text rendering
- One-click meme generation
- Download meme as an image
- Responsive design for desktop, tablet, and mobile
- Fast and lightweight interaction

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
